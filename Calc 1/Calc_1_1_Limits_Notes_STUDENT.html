Limits

Calculus I  ·  §1.1  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

What value does $f(x)$ approach as $x$ gets close to 2 — even if $f(2)$ is undefined? A limit describes the value a function approaches, not necessarily the value it takes at that point. Limits are the foundation on which all of calculus is built.
The Limit

We write:   $\displaystyle\lim_{x \to a} f(x) = L$

if $f(x)$ gets arbitrarily close to $L$ as $x \to a$ from either side.

Key ideas:

  • $x$ approaches $a$ but never equals $a$
  • $f(a)$ may be undefined — the limit can still exist
  • Same value from both sides → limit equals that value
  • Different values from left and right → DNE

Limit vs. function value: “Where is the function heading?” vs. “What does $f$ equal there?” These are independent questions.

Reading Limits from Graphs

Open circle $\circ$ at $(a,\, L)$:
  $f(a)$ undefined, but $\lim_{x\to a}f(x) = L$

Filled circle $\bullet$ at $(a,\, M)$:
  $f(a) = M$  (the actual function value)

The limit depends on the height the curve approaches, not the filled dot.

Removable discontinuity: curve approaches one height (open $\circ$), filled $\bullet$ placed elsewhere → limit = height of the open circle

Jump discontinuity: curve approaches two different heights from left and right → limit = DNE

Worked Examples
Example 1  —  Reading a graph
A graph shows an open circle at $(3,\,5)$ and a filled dot at $(3,\,1)$. Find $\lim_{x \to 3} f(x)$.
Example 2  —  Direct substitution
Find $\lim_{x \to 3}(2x + 1)$.
Example 3  —  Factor and cancel
Find $\displaystyle\lim_{x \to 2}\dfrac{x^2 - 4}{x - 2}$.
Example 4  —  Limit DNE — piecewise
$f(x) = \begin{cases} x+1 & x < 3 \\ x+3 & x \geq 3 \end{cases}$   Find $\lim_{x \to 3}f(x)$.
