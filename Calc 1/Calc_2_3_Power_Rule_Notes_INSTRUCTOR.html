The Power Rule

Calculus I  ·  §2.3  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

The limit definition works but is slow. The Power Rule lets us differentiate polynomials (and much more) in seconds. How far can we push it — and what do we do with roots and negative exponents?
The Power Rule & Basic Rules

$$\frac{d}{dx}\left[x^n\right]=nx^{n-1}$$

Works for any real $n$ — integer, fraction, or negative.

Constant multiple: $\dfrac{d}{dx}[cf(x)]=c\,f'(x)$

Sum/difference: $\dfrac{d}{dx}[f\pm g]=f'\pm g'$

Apply term by term. Constants differentiate to $0$.

▸ Show Proof

We'll prove this for positive integer $n$ using the limit definition.

Start from: $\dfrac{d}{dx}[x^n] = \lim_{h\to 0}\dfrac{(x+h)^n - x^n}{h}$

Expand $(x+h)^n$ using the Binomial Theorem:

$$(x+h)^n = x^n + nx^{n-1}h + \binom{n}{2}x^{n-2}h^2 + \cdots + h^n$$

Every term after $nx^{n-1}h$ contains $h^2$ or higher. Subtract $x^n$ and divide by $h$:

$$\frac{(x+h)^n - x^n}{h} = nx^{n-1} + \binom{n}{2}x^{n-2}h + \cdots + h^{n-1}$$

Every remaining term on the right still contains at least one factor of $h$, so as $h\to 0$ they all vanish. Only $nx^{n-1}$ survives:

$$\lim_{h\to 0}\frac{(x+h)^n - x^n}{h} = nx^{n-1}$$

The same result holds for any real $n$ (fractions, negatives), though those cases require additional techniques beyond this course.

∎

Rewrite first: convert roots to fractional exponents and move terms from denominators before differentiating. $\sqrt{x}=x^{1/2}$, $\dfrac{1}{x^3}=x^{-3}$, $\dfrac{5}{\sqrt{x}}=5x^{-1/2}$.

Using $f'$ — Tangent Lines & Motion

Tangent line at $x=c$:
$y=f'(c)(x-c)+f(c)$

Finding horizontal tangents:
Set $f'(x)=0$ and solve.

Position/velocity:
If $s(t)$ is position, then $s'(t)=v(t)$ is velocity.
If $v(t)$ is velocity, then $v'(t)=a(t)$ is acceleration.

When asked to find where a function is increasing/decreasing, first differentiate, then solve $f'(x)>0$ or $f'(x)<0$.

Worked Examples
Example 1  —  Polynomial — term by term
Differentiate $f(x)=4x^3-7x^2+2x-9$.
$f'(x)=12x^2-14x+2$
Example 2  —  Fractional and negative exponents
Differentiate $f(x)=\sqrt{x}-\dfrac{3}{x^2}$.
Rewrite: $x^{1/2}-3x^{-2}$
$f'(x)=\dfrac{1}{2}x^{-1/2}+6x^{-3}=\dfrac{1}{2\sqrt{x}}+\dfrac{6}{x^3}$
Example 3  —  Tangent line
Find the tangent line to $f(x)=x^3-2x$ at $x=2$.
$f'(x)=3x^2-2$, so $f'(2)=10$; $f(2)=4$
$y=10(x-2)+4=10x-16$
Example 4  —  Horizontal tangent
Find all $x$ where $f(x)=2x^3-9x^2+12x$ has a horizontal tangent.
$f'(x)=6x^2-18x+12=6(x-1)(x-2)=0$
$x=1$ and $x=2$
