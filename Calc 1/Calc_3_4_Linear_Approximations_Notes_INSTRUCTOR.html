Linear Approximations

Calculus I  ·  §3.4  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

Near a point where $f$ is differentiable, the tangent line is an excellent approximation to the curve. How do we use this to estimate values like $\sqrt{4.1}$ or $e^{0.2}$ without a calculator?
Linear Approximation

The linearization of $f$ at $x=a$:

$$L(x)=f(a)+f'(a)(x-a)$$

This is just the tangent line at $a$, reused as an approximation:

$$f(x)\approx L(x) \text{ for } x\text{ near }a$$

Steps:

  1. Choose a nearby $a$ where $f(a)$ is easy
  2. Compute $f(a)$ and $f'(a)$
  3. Write $L(x)=f(a)+f'(a)(x-a)$
  4. Plug in the target $x$ value
Standard Approximations

Near $x=0$ for $e^x$:
$f'(0)=1$, $f(0)=1$: $L(x)=x+1$
$e^{0.2}\approx 1.2$

Near $x=1$ for $\ln x$:
$f'(1)=1$, $f(1)=0$: $L(x)=x-1$
$\ln(1.3)\approx 0.3$

Near $x=4$ for $\sqrt{x}$:
$f'(4)=\frac{1}{4}$, $f(4)=2$: $L(x)=2+\frac{x-4}{4}$
$\sqrt{4.4}\approx 2.1$

Worked Examples
Example 1  —  Approximate $\sqrt{4.1}$
Use $f(x)=\sqrt{x}$ near $a=4$ to estimate $\sqrt{4.1}$.
$f'(x)=\frac{1}{2\sqrt{x}}$, $f'(4)=\frac{1}{4}$, $f(4)=2$
$L(x)=2+\frac{1}{4}(x-4)$
$L(4.1)=2+\frac{0.1}{4}=2+0.025=2.025$
Example 2  —  Approximate $e^{0.1}$
Use $f(x)=e^x$ near $a=0$ to estimate $e^{0.1}$.
$f'(0)=1$, $f(0)=1$; $L(x)=1+x$
$e^{0.1}\approx L(0.1)=1.1$
Example 3  —  Approximate $\ln(0.9)$
Use $f(x)=\ln x$ near $a=1$ to estimate $\ln(0.9)$.
$f'(1)=1$, $f(1)=0$; $L(x)=x-1$
$\ln(0.9)\approx L(0.9)=0.9-1=-0.1$
Example 4  —  Write the linearization
Find $L(x)$ for $f(x)=x^3$ at $a=2$.
$f'(x)=3x^2$, $f'(2)=12$, $f(2)=8$
$L(x)=8+12(x-2)=12x-16$
Practice Problems
1.
Find $L(x)$ for $f(x)=x^2$ at $a=3$.
$f'(3)=6$, $f(3)=9$. $L(x)=9+6(x-3)=6x-9$
2.
Use $L(x)$ above to estimate $3.1^2$.
$L(3.1)=6(3.1)-9=18.6-9=9.6$
3.
Find $L(x)$ for $f(x)=e^x$ at $a=0$, and estimate $e^{0.3}$.
$L(x)=1+x$; $e^{0.3}\approx 1.3$
4.
Find $L(x)$ for $f(x)=\sqrt{x}$ at $a=9$.
$f'(9)=\frac{1}{6}$, $f(9)=3$. $L(x)=3+\frac{x-9}{6}$
5.
Use $L(x)$ above to estimate $\sqrt{9.3}$.
$L(9.3)=3+0.3/6=3+0.05=3.05$
6.
Estimate $\ln(1.1)$ using $f(x)=\ln x$ near $a=1$.
$L(x)=x-1$; $\ln(1.1)\approx 0.1$
