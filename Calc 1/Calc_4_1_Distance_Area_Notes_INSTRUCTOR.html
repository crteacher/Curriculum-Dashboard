Distance Traveled & Area Under a Curve

Calculus I  ·  §4.1  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

If you know your speed at every moment, how do you find total distance traveled? And geometrically, what does $\displaystyle\int_a^b f(x)\,dx$ actually represent? We start with the big picture: area as accumulated change.
Area = Accumulated Change

$$\int_a^b f(x)\,dx = \text{area under } f \text{ from } a \text{ to } b$$

If $f(x)$ is a rate of change (e.g., speed), the integral gives total change (distance).

For simple shapes, use geometry:
rectangle: $\text{base}\times\text{height}$
triangle: $\frac{1}{2}\cdot\text{base}\times\text{height}$
semicircle: $\frac{1}{2}\pi r^2$

▸ Geometric Argument

Here's why area under a rate curve equals total change — let's think about it with velocity.

Imagine your speed $v(t)$ is constant at some value $v_0$ on an interval $[a,b]$. Distance = speed × time = $v_0\cdot(b-a)$. Geometrically, that's the area of a rectangle with height $v_0$ and width $(b-a)$.

Now suppose speed varies. Break $[a,b]$ into tiny slices of width $dt$. On each slice, speed barely changes, so distance $\approx v(t)\cdot dt$ — a thin rectangle of height $v(t)$ and width $dt$.

Adding all those rectangles: total distance $\approx \sum v(t)\,dt$. As the slices get infinitely thin, the sum becomes the integral $\int_a^b v(t)\,dt$, which equals the exact area under the curve.

The same logic applies to any rate of change: the integral $\int_a^b f(x)\,dx$ measures total accumulated change, which is the area trapped between the curve and the $x$-axis.

∎

Left Riemann sum with 3 rectangles — underestimates area under an increasing curve
Riemann Sums from a Table

Approximate $\displaystyle\int_a^b f(x)\,dx$ by rectangles:

Left sum: use $f$ at the left endpoint of each subinterval
Right sum: use $f$ at the right endpoint

$$\text{Total}\approx\sum f(x_i)\cdot\Delta x_i$$

The width $\Delta x_i$ is the gap between consecutive $x$-values (may vary if the table is unequal).

▸ Geometric Argument

The key idea is simple: replace a curved region with rectangles whose area we can compute exactly.

Divide $[a,b]$ into subintervals. On each subinterval $[x_i, x_{i+1}]$, choose a sample point and draw a rectangle with height $f(x_i)$ and width $\Delta x_i = x_{i+1}-x_i$.

Each rectangle's area is $f(x_i)\cdot\Delta x_i$ (base times height). Summing all rectangles: $$\text{Total area}\approx\sum f(x_i)\cdot\Delta x_i$$

As the widths $\Delta x_i$ shrink toward zero, the jagged rectangle outline hugs the curve more and more tightly. In the limit, the sum equals the exact area — which is the definition of $\int_a^b f(x)\,dx$.

∎

When the function is positive, both left and right sums give positive area. When the function dips below zero, rectangles contribute negative area.

Worked Examples
Example 1  —  Left sum from a table
Velocity data: $t=0,1,2,3$, $v=2,5,4,6$ mph. Estimate distance using a left Riemann sum.
Left sum $= v(0)\cdot1+v(1)\cdot1+v(2)\cdot1$
$=2+5+4=11$ miles
Example 2  —  Right sum from a table
Same data. Estimate distance using a right Riemann sum.
Right sum $= v(1)\cdot1+v(2)\cdot1+v(3)\cdot1$
$=5+4+6=15$ miles
Example 3  —  Geometric area
Find $\displaystyle\int_0^3 2\,dx$ using geometry.
Rectangle: base $=3$, height $=2$
Area $= 3\times 2=6$
Example 4  —  Triangle area
Find $\displaystyle\int_0^4 x\,dx$ using geometry.
Triangle: base $=4$, height $=4$
Area $=\dfrac{1}{2}\cdot 4\cdot 4=8$
