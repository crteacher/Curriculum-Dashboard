Concavity & 2nd Derivative Test

Calculus I  ·  §3.6.5  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

The first derivative tells us where $f$ goes up or down. The second derivative tells us whether the graph curves up like a cup or down like a hill. Can we use $f''$ to classify critical numbers more quickly?
Concavity & Inflection Points

$f''(x)>0$ on interval → $f$ is concave up ∪
$f''(x)<0$ on interval → $f$ is concave down ∩

An inflection point is where $f''$ changes sign.
To find: set $f''(x)=0$, check that the sign actually changes.

Concave up means the tangent line lies below the curve. Concave down means it lies above. An inflection point is NOT just where $f''=0$ — the sign must change.

Concave up (cup): tangent lies below; concave down (hill): tangent lies above
Second Derivative Test

If $f'(c)=0$:

  • $f''(c)>0$ → concave up at $c$ → local min
  • $f''(c)<0$ → concave down at $c$ → local max
  • $f''(c)=0$ → inconclusive (use FDT instead)

Memory: concave up = cup shape = min at bottom.

The 2nd Derivative Test is often faster than FDT when $f''$ is easy to compute. But if $f''(c)=0$, you must fall back to the First Derivative Test.

Worked Examples
Example 1  —  Concavity intervals
Find where $f(x)=x^3-3x^2$ is concave up/down.
$f'(x)=3x^2-6x$, $f''(x)=6x-6$
$f''=0$ at $x=1$
$x<1$: $f''<0$ → concave down
$x>1$: $f''>0$ → concave up
Example 2  —  Inflection point
Find the inflection point of $f(x)=x^3-3x^2$.
$f''$ changes sign at $x=1$: inflection point at $(1,f(1))=(1,-2)$
Example 3  —  Second Derivative Test
Classify the critical numbers of $f(x)=x^3-3x^2$ using the 2nd Derivative Test.
Critical: $x=0$ and $x=2$
$f''(0)=-6<0$ → local max
$f''(2)=6>0$ → local min
Example 4  —  Inconclusive case
Apply 2nd Derivative Test to $f(x)=x^4$ at $x=0$.
$f'(x)=4x^3=0$ at $x=0$
$f''(x)=12x^2$, $f''(0)=0$ → inconclusive
FDT: $f'$ changes $-\to+$ → local min
Practice Problems
1.
Find the concavity intervals for $f(x)=x^4-6x^2$.
$f''=12x^2-12=0$ at $x=\pm 1$. CU on $(-\infty,-1)$ and $(1,\infty)$; CD on $(-1,1)$
2.
Find inflection points of $f(x)=x^4-6x^2$.
$x=\pm 1$; points $(\pm 1,-5)$
3.
Classify $x=0$ for $f(x)=x^4-6x^2$ using SDT.
$f'(0)=0$, $f''(0)=-12<0$ → local max
4.
Classify $x=\pm\sqrt{3}$ for $f(x)=x^4-6x^2$ using SDT.
$f''(\sqrt{3})=24>0$ → local min; same at $-\sqrt{3}$
5.
Where is $f(x)=e^{-x^2}$ concave up?
$f''=e^{-x^2}(4x^2-2)>0$ when $|x|>1/\sqrt{2}$
6.
Does $f(x)=x^3$ have an inflection point? Where?
$f''=6x=0$ at $x=0$; sign changes → inflection at $(0,0)$
