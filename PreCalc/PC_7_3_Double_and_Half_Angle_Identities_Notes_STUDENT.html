Double and Half Angle Identities

Pre-Calculus  ·  §7.3  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

$\sin(2x)$ and $2\sin(x)$ are not the same function. Can you derive the formula for $\sin(2x)$ from the sum formula you already know?
Double Angle Formulas

Derived from sum formulas with $A=B=\theta$:

$\sin(2\theta)=2\sin\theta\cos\theta$

$\cos(2\theta)=\cos^2\theta-\sin^2\theta$
$\phantom{\cos(2\theta)}=1-2\sin^2\theta$
$\phantom{\cos(2\theta)}=2\cos^2\theta-1$

$\tan(2\theta)=\dfrac{2\tan\theta}{1-\tan^2\theta}$

Half Angle Formulas

Solve $\cos(2\theta)=2\cos^2\theta-1$ for $\cos\theta$:

$\cos\dfrac{\theta}{2}=\pm\sqrt{\dfrac{1+\cos\theta}{2}}$

$\sin\dfrac{\theta}{2}=\pm\sqrt{\dfrac{1-\cos\theta}{2}}$

$\tan\dfrac{\theta}{2}=\dfrac{\sin\theta}{1+\cos\theta}=\dfrac{1-\cos\theta}{\sin\theta}$

Sign chosen by the quadrant of $\theta/2$.

Worked Examples
Example 1 — Apply double angle
If $\sin\theta=3/5$ ($\theta$ in QI), find $\sin(2\theta)$ and $\cos(2\theta)$.
$\cos\theta=4/5$. $\sin(2\theta)=2(3/5)(4/5)=24/25$. $\cos(2\theta)=(16/25)-(9/25)=7/25$.
Example 2 — Exact value with double angle
Evaluate $\cos^2(15°)-\sin^2(15°)$.
$\cos(2\cdot15°)=\cos30°=\dfrac{\sqrt3}{2}$.
Example 3 — Half angle
Use the half-angle formula to find $\sin22.5°$ exactly.
$\sin22.5°=\sin\dfrac{45°}{2}=\sqrt{\dfrac{1-\cos45°}{2}}=\sqrt{\dfrac{1-\sqrt2/2}{2}}=\dfrac{\sqrt{2-\sqrt2}}{2}$.
Example 4 — Simplify
Simplify $\dfrac{\sin(2x)}{2\cos x}$.
$\dfrac{2\sin x\cos x}{2\cos x}=\sin x$ (for $\cos x\neq0$).
Practice Problems
1.
Find $\sin(2\theta)$ if $\cos\theta=-4/5$ and $\theta$ in QIII.
$\sin\theta=-3/5$; $\sin(2\theta)=2(-3/5)(-4/5)=24/25$.
2.
Find $\cos(2x)$ if $\sin x=1/3$.
$1-2(1/9)=7/9$.
3.
Evaluate $2\cos^2(75°)-1$.
$\cos(150°)=-\sqrt3/2$.
4.
Use half-angle: find $\cos22.5°$.
$\sqrt{\dfrac{1+\cos45°}{2}}=\dfrac{\sqrt{2+\sqrt2}}{2}$.
5.
Simplify $\cos^2\theta-\sin^2\theta$.
$\cos(2\theta)$.
6.
Find $\tan(2\theta)$ if $\tan\theta=2$.
$\dfrac{4}{1-4}=-\dfrac{4}{3}$.
