Core Ideas
Piecewise Functions A piecewise function uses different formulas on different parts of the domain. To evaluate $f(a)$: identify which condition $a$ satisfies, then apply that formula. The boundary value belongs to whichever piece uses $\leq$ or $\geq$ (closed condition).
Domain of a Piecewise Function The domain is the union of all the sub-domains listed. If every real number is covered (e.g., $x<0$ and $x\geq 0$), the domain is $(-\infty,\infty)$. Gaps in the conditions create gaps in the domain.
Continuity at a Boundary A piecewise function is continuous at $x=c$ if the left-side limit equals the right-side limit equals $f(c)$. Check: compute the value from each piece at the boundary and see if they agree.
