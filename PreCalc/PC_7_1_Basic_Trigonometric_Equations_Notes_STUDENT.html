Basic Trigonometric Equations

Pre-Calculus  ·  §7.1  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

$\sin\theta=0.5$ has infinitely many solutions — $\theta=30°, 150°, 390°, \ldots$ How do we find them all, and how do we restrict to just one period $[0,2\pi)$?
Solving Basic Trig Equations

Steps:

  1. Isolate the trig function.
  2. Find the reference angle using inverse trig.
  3. Use ASTC to find all solutions in $[0,2\pi)$.
  4. Add $2\pi k$ for all solutions (if asked).

For $\tan$: add $\pi k$ (period is $\pi$).

Solutions in $[0,2\pi)$ by Function

$\sin\theta=c$: solutions in QI and QII if $c>0$; QIII and QIV if $c<0$.

$\cos\theta=c$: solutions in QI and QIV if $c>0$; QII and QIII if $c<0$.

$\tan\theta=c$: solutions in QI and QIII if $c>0$; QII and QIV if $c<0$.

Special values to memorize: $\sin,\cos\in\{0,\pm\tfrac{1}{2},\pm\tfrac{\sqrt2}{2},\pm\tfrac{\sqrt3}{2},\pm1\}$.

Worked Examples
Example 1 — Basic sine equation
Solve $\sin\theta=\dfrac{\sqrt3}{2}$ on $[0,2\pi)$.
Reference angle $\pi/3$. QI: $\theta=\pi/3$; QII: $\theta=\pi-\pi/3=2\pi/3$.
Example 2 — Negative cosine
Solve $\cos\theta=-\dfrac{\sqrt2}{2}$ on $[0,2\pi)$.
Reference $\pi/4$. Cos negative in QII, QIII: $\theta=3\pi/4,5\pi/4$.
Example 3 — Tangent equation
Solve $\tan\theta=1$ on $[0,2\pi)$.
Reference $\pi/4$. Tan positive in QI, QIII: $\theta=\pi/4,5\pi/4$.
Example 4 — Equation requiring algebra
Solve $2\sin\theta+\sqrt3=0$ on $[0,2\pi)$.
$\sin\theta=-\dfrac{\sqrt3}{2}$. Reference $\pi/3$. Sin negative in QIII, QIV: $\theta=\pi+\pi/3=4\pi/3$ and $\theta=2\pi-\pi/3=5\pi/3$.
Practice Problems
1.
Solve $\sin\theta=1$ on $[0,2\pi)$.
$\theta=\pi/2$.
2.
Solve $\cos\theta=0$ on $[0,2\pi)$.
$\theta=\pi/2,3\pi/2$.
3.
Solve $\tan\theta=-\sqrt3$ on $[0,2\pi)$.
Reference $\pi/3$; QII, QIV: $\theta=2\pi/3,5\pi/3$.
4.
Solve $2\cos\theta=1$ on $[0,2\pi)$.
$\cos\theta=1/2$; $\theta=\pi/3,5\pi/3$.
5.
Solve $\sin\theta=-1/2$ on $[0,2\pi)$.
$\theta=7\pi/6,11\pi/6$.
6.
Write general solution of $\cos\theta=0$.
$\theta=\pi/2+n\pi$, $n$ integer.
