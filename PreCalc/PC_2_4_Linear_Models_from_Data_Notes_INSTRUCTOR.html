Linear Models from Data

Pre-Calculus  ·  §2.4  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

If you measure the temperature and cricket chirp rate on 10 different nights, how would you find the best single line that summarizes the relationship?
Scatter Plots and Correlation

A scatter plot plots data as individual points $(x_i,y_i)$.

Correlation coefficient $r$:
$r\approx 1$: strong positive linear trend.
$r\approx -1$: strong negative linear trend.
$r\approx 0$: little or no linear trend.

$R^2$ (coefficient of determination): how well the line fits. $R^2$ close to 1 = good fit.

▸ Common Mistake

Students treat $r$ and $R^2$ as interchangeable — they're not. $r=-0.9$ and $r=0.9$ describe equally strong relationships, just in opposite directions, but $R^2=0.81$ for both, since $R^2=r^2$ throws away the sign. Reading $r=-0.9$ as "a bad fit" because it's negative confuses direction with strength.

This gets worse the other way too: a student who sees $R^2=0.81$ and reports "$r=0.81$" has lost information — there are two possible values of $r$ ($\pm0.9$) consistent with that $R^2$, and only the scatter plot's direction tells you which one is real.

Line of Best Fit

The line of best fit (least-squares regression line) minimizes the sum of squared residuals.

Use a graphing calculator or spreadsheet (Google Sheets: Insert → Chart → Trendline → Linear).

Residual = actual $y$ − predicted $y$. Good fits have small residuals.

Use the model to interpolate (within data range) or extrapolate (outside range — use caution).

▸ Common Mistake

Students compute residuals as predicted $y$ minus actual $y$ — backwards from the definition. The order matters because the sign tells you which way the model is wrong: a residual of actual $-$ predicted $= +3$ means the real data point sits above the line (the model underestimated); flip the subtraction and that same point reads as $-3$, which says the opposite.

The mix-up is easy to make because both quantities are "the difference between two numbers," but only one order matches the convention used to interpret residual plots and to define $R^2$ — so getting it backwards silently flips every conclusion you draw about over- or under-prediction.

Worked Examples
Example 1 — Interpret a scatter plot
A scatter plot of study hours vs. test scores shows points generally rising from left to right. Describe the correlation.
Positive correlation — more study hours tend to correspond with higher scores. $r>0$.
Example 2 — Use a regression line
A regression line for cricket chirps vs. temperature is $T=0.25n+40$ where $n$ is chirps/min. Predict $T$ when $n=100$.
$T=0.25(100)+40=65°$.
Example 3 — Interpret $R^2$
A linear regression gives $R^2=0.92$. What does this mean?
About 92% of the variation in the dependent variable is explained by the linear model. This is a strong fit.
Example 4 — Identify outliers
Most data follow a linear trend except one point far from the line. How does removing it affect the model?
Removing an outlier often increases $R^2$ and changes the slope/intercept. Always note when outliers are removed.
Practice Problems
1.
Describe the correlation: as $x$ increases, $y$ decreases strongly.
Strong negative correlation, $r\approx -1$.
2.
A regression line is $y=3.2x+11.5$. Predict $y$ when $x=4$.
$y=3.2(4)+11.5=24.3$.
3.
What does $r=0.1$ indicate about a scatter plot?
Very weak linear relationship between the variables.
4.
$R^2=0.45$ for a linear model. Is this a strong fit?
No — only 45% of variation is explained. A moderate-to-weak fit.
5.
A data set of (hours worked, income) has $r=0.98$. Interpret this.
Very strong positive linear relationship between hours and income.
6.
Why is extrapolating far beyond the data range risky?
The linear trend may not continue — the model was only validated within the observed data range.
