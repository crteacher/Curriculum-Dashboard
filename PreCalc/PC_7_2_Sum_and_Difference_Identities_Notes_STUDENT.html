Sum and Difference Identities

Pre-Calculus  ·  §7.2  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

Can you find $\cos75°$ exactly without a calculator? If $75°=45°+30°$, is $\cos75°=\cos45°+\cos30°$? (Spoiler: it's not — so what is the correct formula?)
Sum and Difference Formulas

Sine:
$\sin(A+B)=\sin A\cos B+\cos A\sin B$
$\sin(A-B)=\sin A\cos B-\cos A\sin B$

Cosine:
$\cos(A+B)=\cos A\cos B-\sin A\sin B$
$\cos(A-B)=\cos A\cos B+\sin A\sin B$

Tangent:
$\tan(A\pm B)=\dfrac{\tan A\pm\tan B}{1\mp\tan A\tan B}$

Applications

Exact values: write angle as sum/difference of special angles ($30°,45°,60°,90°,\ldots$).

Verify identities and simplify expressions using these formulas.

Note: $\cos$ eats negatives ($\cos(-x)=\cos x$). $\sin$ flips sign ($\sin(-x)=-\sin x$).

Worked Examples
Example 1 — Exact value
Find $\cos75°$ exactly.
$\cos(45°+30°)=\cos45°\cos30°-\sin45°\sin30°=\dfrac{\sqrt2}{2}\cdot\dfrac{\sqrt3}{2}-\dfrac{\sqrt2}{2}\cdot\dfrac{1}{2}=\dfrac{\sqrt6-\sqrt2}{4}$.
Example 2 — Exact value — sine
Find $\sin15°$.
$\sin(45°-30°)=\sin45°\cos30°-\cos45°\sin30°=\dfrac{\sqrt6-\sqrt2}{4}$.
Example 3 — Verify
Verify $\cos(a-b)=\cos a\cos b+\sin a\sin b$ for $a=\pi/3$, $b=\pi/6$.
LHS: $\cos(\pi/6)=\sqrt3/2$. RHS: $\cos(\pi/3)\cos(\pi/6)+\sin(\pi/3)\sin(\pi/6)=(1/2)(\sqrt3/2)+(\sqrt3/2)(1/2)=\sqrt3/2$ ✓.
Example 4 — Simplify
Simplify $\cos(2x)\cos(x)+\sin(2x)\sin(x)$.
$\cos(2x-x)=\cos x$.
Practice Problems
1.
Find $\sin75°$ exactly.
$\sin(45°+30°)=\dfrac{\sqrt6+\sqrt2}{4}$.
2.
Find $\cos15°$ exactly.
$\cos(45°-30°)=\dfrac{\sqrt6+\sqrt2}{4}$.
3.
Expand $\sin(x+\pi)$.
$\sin x\cos\pi+\cos x\sin\pi=-\sin x$.
4.
Simplify $\sin(A+B)-\sin(A-B)$.
$2\cos A\sin B$.
5.
Find $\tan75°$ exactly.
$\dfrac{\tan45°+\tan30°}{1-\tan45°\tan30°}=\dfrac{1+1/\sqrt3}{1-1/\sqrt3}=2+\sqrt3$.
6.
If $\sin A=3/5$ ($A$ in QI) and $\cos B=5/13$ ($B$ in QI), find $\sin(A+B)$.
$\sin A\cos B+\cos A\sin B=(3/5)(5/13)+(4/5)(12/13)=63/65$.
