Logarithmic Functions

Pre-Calculus  ·  §4.3  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

Exponential functions grow — but how do we solve $3^x=50$ for $x$ exactly? What operation undoes an exponential?
Logarithms as Inverses

Definition: $\log_b y = x \iff b^x = y$  ($b>0$, $b\neq1$, $y>0$).

Common log: $\log x = \log_{10}x$
Natural log: $\ln x = \log_e x$

Key values: $\log_b 1=0$; $\log_b b=1$; $\log_b(b^x)=x$; $b^{\log_b x}=x$.

Graph: domain $(0,\infty)$, range $(-\infty,\infty)$, VA at $x=0$, passes through $(1,0)$.

Properties of Logarithms
  • Product: $\log_b(MN)=\log_b M+\log_b N$
  • Quotient: $\log_b\!\left(\dfrac{M}{N}\right)=\log_b M-\log_b N$
  • Power: $\log_b(M^p)=p\log_b M$
  • Change of base: $\log_b M=\dfrac{\ln M}{\ln b}=\dfrac{\log M}{\log b}$
Worked Examples
Example 1 — Convert between forms
Rewrite as exponentials: (a) $\log_3 81=4$ (b) $\ln 1=0$ (c) $\log_5 x=3$.
(a) $3^4=81$. (b) $e^0=1$. (c) $5^3=x$, so $x=125$.
Example 2 — Evaluate
Find: (a) $\log_2 32$ (b) $\log 1000$ (c) $\ln e^5$.
(a) $2^5=32\Rightarrow 5$. (b) $10^3=1000\Rightarrow 3$. (c) $5$.
Example 3 — Expand using properties
Expand $\log\!\left(\dfrac{x^3\sqrt{y}}{z^2}\right)$.
$3\log x+\tfrac{1}{2}\log y-2\log z$.
Example 4 — Change of base
Evaluate $\log_5 100$ using change of base.
$\dfrac{\log 100}{\log 5}=\dfrac{2}{0.699}\approx 2.86$.
Practice Problems
1.
Rewrite $2^6=64$ in logarithmic form.
$\log_2 64=6$.
2.
Evaluate $\log_3 \dfrac{1}{27}$.
$3^{-3}=\tfrac{1}{27}\Rightarrow -3$.
3.
Expand $\ln(x^2 y^3)$.
$2\ln x+3\ln y$.
4.
Condense $\log 4+\log 6 - \log 3$.
$\log\tfrac{24}{3}=\log 8$.
5.
Evaluate $\log_7 49$.
$7^2=49\Rightarrow 2$.
6.
State domain of $f(x)=\log(2x-6)$.
$2x-6>0\Rightarrow x>3$; domain $(3,\infty)$.
