Graphing Linear Equations

PreCalc  ·  §2.2  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Slope-intercept form $y = mx + b$ — $m$ is the slope; $b$ is the y-intercept.
The y-intercept is the point $(0,\,b)$ where the line crosses the y-axis.
Plot it, then use slope (rise/run) to find a second point.
Intercepts y-intercept: set $x=0$  →  solve for $y$  →  point $(0,\,y)$.
x-intercept: set $y=0$  →  solve for $x$  →  point $(x,\,0)$.
Both intercepts are written as ordered pairs, e.g. $(0,5)$ or $(-3,0)$.
Standard form $Ax + By = C$  →  rewrite as $y = -\dfrac{A}{B}x + \dfrac{C}{B}$ to read off slope and y-intercept.
For the x-intercept, set $y=0$:   $x = \dfrac{C}{A}$.
Practice Problems
