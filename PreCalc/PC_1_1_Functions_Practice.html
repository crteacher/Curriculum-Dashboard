Functions

PreCalc  ·  §1.1  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Function Definition A function $f$ is a rule that assigns each input $x$ in the domain exactly one output $f(x)$ in the range. The vertical line test: a graph represents a function if and only if every vertical line crosses it at most once.
Domain and Range The domain is the set of all valid inputs; the range is the set of all resulting outputs. For algebraic functions: exclude $x$-values that cause division by zero or a negative under an even root.
Function Notation $f(a)$ means "evaluate $f$ at $x = a$": substitute $a$ for every $x$. For example, if $f(x)=x^2-3x$, then $f(2)=4-6=-2$.
