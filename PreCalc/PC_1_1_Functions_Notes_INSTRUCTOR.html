Functions

Pre-Calculus  ·  §1.1  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

A vending machine takes your dollar and gives you a snack — the same snack every time you press the same button. Is that relationship a function? What property makes a relation a function?
Relations and Functions

Relation: a rule that associates elements of one set (domain) with elements of another set (range).

Function: a relation where every input $x$ is associated with exactly one output $y$.

Notation: $y = f(x)$ — read "$f$ of $x$". The input $x$ is the independent variable; $y$ is the dependent variable.

▸ Common Mistake

"Exactly one output per input" restricts inputs, not outputs. Students often disqualify a relation because an output repeats — calling $\{(1,5),(2,5)\}$ "not a function" because $5$ shows up twice. That's perfectly fine: two different inputs are allowed to share an output. What breaks the rule is one input with two outputs, like $\{(1,2),(1,3)\}$. (Forbidding repeated outputs is a different, stronger property — being one-to-one.)

Tests: Vertical Line Test — a graph represents a function if every vertical line hits it at most once.

Evaluating Functions

To evaluate $f(a)$: substitute $a$ for every $x$ in the formula.

Example: if $f(x)=x^2-3x+1$, then
$f(2)=4-6+1=-1$
$f(-x)=x^2+3x+1$
$f(x+h)=(x+h)^2-3(x+h)+1$

▸ Common Mistake

$f$ is the name of a rule, not a quantity multiplying its input, so it does not distribute over addition: $f(x+h)\neq f(x)+h$ and $f(x+h)\neq f(x)+f(h)$. To find $f(x+h)$ you substitute the whole quantity $x+h$ for every $x$ in the formula. For $f(x)=x^2-3x+1$ that means $f(x+h)=(x+h)^2-3(x+h)+1$, then expand — you can't just tack $+h$ onto $f(x)$. This is the slip that wrecks difference-quotient problems.

Three representations: equation, graph, table. Use the vertical-line test on graphs; check for repeated inputs in tables.

Worked Examples
Example 1 — Is it a function?
Which relations are functions? (a) $\{(1,2),(2,3),(3,4)\}$   (b) $\{(1,2),(1,3)\}$   (c) $y=x^2$   (d) $x=y^2$
(a) Yes — each input maps to one output. (b) No — input 1 maps to two outputs. (c) Yes — VLT passes. (d) No — e.g. $x=4$ gives $y=\pm2$.
Example 2 — Evaluating from a formula
Let $f(x)=2x^2-x+3$. Find $f(0)$, $f(-2)$, and $f(x+1)$.
$f(0)=3$; $f(-2)=2(4)+2+3=13$; $f(x+1)=2(x+1)^2-(x+1)+3=2x^2+3x+4$.
Example 3 — Evaluating from a table
A table shows: $x$: 0,1,2,3 and $f(x)$: 5,3,1,−1. Find $f(2)$ and solve $f(x)=-1$.
$f(2)=1$; $f(x)=-1$ when $x=3$.
Example 4 — Difference quotient
For $f(x)=x^2+1$, simplify $\dfrac{f(x+h)-f(x)}{h}$, $h\neq 0$.
$\dfrac{(x+h)^2+1-(x^2+1)}{h}=\dfrac{2xh+h^2}{h}=2x+h$.
Practice Problems
1.
State the domain and range of $\{(0,1),(2,3),(4,5)\}$. Is it a function?
Domain $\{0,2,4\}$, Range $\{1,3,5\}$. Yes, it is a function.
2.
Let $f(x)=3x-5$. Find $f(4)$ and $f(-1)$.
$f(4)=7$; $f(-1)=-8$.
3.
Let $g(x)=x^2-4$. Find $g(0)$, $g(2)$, $g(-3)$.
$g(0)=-4$; $g(2)=0$; $g(-3)=5$.
4.
Does $x^2+y^2=9$ define $y$ as a function of $x$? Explain.
No — for most $x$ values there are two $y$ values (e.g. $x=0$ gives $y=\pm3$).
5.
For $f(x)=2x+1$, simplify $\dfrac{f(x+h)-f(x)}{h}$.
$\dfrac{2(x+h)+1-(2x+1)}{h}=\dfrac{2h}{h}=2$.
6.
Given the graph of $f$, explain how to use the vertical line test.
If any vertical line intersects the graph more than once, the graph does not represent a function.
