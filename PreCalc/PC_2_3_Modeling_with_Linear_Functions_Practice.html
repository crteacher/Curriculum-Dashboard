Modeling with Linear Functions

PreCalc  ·  §2.3  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Building a linear model A linear model has the form $f(x) = mx + b$, where:
• $m$ = rate of change (slope) — how much the output changes per unit increase in $x$.
• $b$ = initial value (y-intercept) — the output when $x = 0$.
Given two data points, find $m$ first, then $b = y - mx$.
Intersection of two models Two linear models $f(x)$ and $g(x)$ intersect where $f(x) = g(x)$.
Set the expressions equal and solve for $x$ — this is the break-even or equilibrium quantity.
