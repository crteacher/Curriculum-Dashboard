Graphing Linear Equations

Pre-Calculus  ·  §2.2  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

A hot-air balloon starts 1 m off the ground and rises at 2 m/s. If you graph height vs. time, what does the $y$-intercept represent? What does the slope represent?
Graphing from Slope and Intercepts

Using slope-intercept form $y=mx+b$:
Plot the $y$-intercept $(0,b)$, then use rise/run to find a second point.

Finding intercepts:
$y$-intercept: set $x=0$, solve for $y$.
$x$-intercept: set $y=0$, solve for $x$.

Special lines:
$x=a$ — vertical; undefined slope.
$y=b$ — horizontal; slope $=0$.

Parallel and Perpendicular Lines

Parallel: same slope, different intercept. Graphs never intersect.

Perpendicular: slopes multiply to $-1$. Graphs cross at $90^\circ$.

To find a parallel/perpendicular line: identify the slope of the given line, then use point-slope form with the new point.

Always check your answer by verifying the new line passes through the required point.

Worked Examples
Example 1 — Graph using intercepts
Graph $2x+3y=12$ by finding $x$- and $y$-intercepts.
$x$-int: set $y=0\Rightarrow x=6$, point $(6,0)$. $y$-int: set $x=0\Rightarrow y=4$, point $(0,4)$. Draw the line.
Example 2 — Graph from slope and point
Graph the line through $(3,2)$ with slope $m=-\tfrac{4}{5}$.
Plot $(3,2)$. Go down 4, right 5 to get $(8,-2)$. Draw line through both points.
Example 3 — Parallel line
Find and sketch the line through $(1,-3)$ parallel to $2x+2y=6$.
Given slope: $-1$. Parallel: $y+3=-(x-1)\Rightarrow y=-x-2$.
Example 4 — Perpendicular line
Find the line through $(1,-2)$ perpendicular to $x+3y=6$.
Slope of given: $-\tfrac{1}{3}$. Perpendicular slope: $3$. Line: $y=3x-5$.
Practice Problems
1.
Find the $x$- and $y$-intercepts of $3x-4y=12$.
$x$-int: $(4,0)$; $y$-int: $(0,-3)$.
2.
Graph $y=-2x+5$. Label the intercepts.
$y$-int $(0,5)$, $x$-int $(2.5,0)$.
3.
Write the equation of the vertical line through $(4,0)$.
$x=4$.
4.
Find the line through $(0,3)$ parallel to $y=-x+7$.
$y=-x+3$.
5.
Find the line through $(2,1)$ perpendicular to $y=\tfrac{1}{2}x+3$.
Slope $-2$: $y-1=-2(x-2)\Rightarrow y=-2x+5$.
6.
A balloon rises from 1 m at 2 m/s. Write $h(t)$ and find $h(10)$.
$h(t)=2t+1$; $h(10)=21$ m.
