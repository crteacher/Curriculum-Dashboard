Core Ideas
Finding Domain Algebraically Exclude values that cause: (1) division by zero ŌĆö set denominator $\neq 0$; (2) even root of a negative ŌĆö set radicand $\geq 0$. Write the remaining values in interval notation.
Interval Notation Use $[$ or $]$ for included endpoints, $($ or $)$ for excluded endpoints or $\pm\infty$. Union $\cup$ joins disjoint pieces: $(-\infty,2)\cup(2,\infty)$ means all reals except $2$.
Range from a Formula For $f(x)=\sqrt{x-a}+k$: range is $[k,\infty)$. For $f(x)=\frac{1}{x-a}+k$: range is $(-\infty,k)\cup(k,\infty)$. For quadratics $f(x)=a(x-h)^2+k$: range is $[k,\infty)$ if $a>0$, $(-\infty,k]$ if $a<0$.
