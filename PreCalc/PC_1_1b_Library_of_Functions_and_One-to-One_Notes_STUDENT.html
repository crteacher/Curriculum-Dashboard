Library of Functions & One-to-One

Pre-Calculus  ·  §1.1 Pt 2  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

The square function $f(x)=x^2$ sends both $x=2$ and $x=-2$ to the output 4. Does that cause any problems? What would we need to "undo" a function?
Library of Basic Functions

Know these graphs, domain, range, and key points at $x=-1,0,1$:

  • Constant: $y=b$ — range $\{b\}$
  • Linear: $y=x$ — domain/range $(-\infty,\infty)$
  • Quadratic: $y=x^2$ — range $[0,\infty)$
  • Cubic: $y=x^3$ — domain/range $(-\infty,\infty)$
  • Square root: $y=\sqrt{x}$ — domain $[0,\infty)$
  • Reciprocal: $y=1/x$ — domain $x\neq 0$
  • Absolute value: $y=|x|$ — range $[0,\infty)$
One-to-One Functions

One-to-one (1-1): a function where different inputs always give different outputs — if $x_1\neq x_2$ then $f(x_1)\neq f(x_2)$.

Horizontal Line Test: a function is 1-1 iff every horizontal line hits its graph at most once.

$y=x^3$: 1-1 ✓   $y=x^2$: not 1-1 ✗ (fails HLT)

Only 1-1 functions have inverses. Knowing your library helps you identify which functions qualify.

Worked Examples
Example 1 — Identify domain and range
State the domain and range of $f(x)=\sqrt{x-2}$.
Domain: $x-2\geq 0\Rightarrow x\geq 2$, i.e. $[2,\infty)$. Range: $[0,\infty)$.
Example 2 — Key points of library functions
For $f(x)=x^3$ and $g(x)=|x|$, find the value at $x=-1,0,1$.
$f$: $-1,0,1$. $g$: $1,0,1$.
Example 3 — Horizontal line test
Is $f(x)=x^2-4$ one-to-one? Justify.
No. E.g. $f(2)=0=f(-2)$. Horizontal line $y=0$ hits the graph twice.
Example 4 — Identify the function from its graph
A graph passes through $(0,0)$, is always $\geq 0$, and is not linear. Which library function is it most likely?
Either $y=x^2$ (U-shape, smooth) or $y=|x|$ (V-shape, corner at origin). Distinguish by whether there is a sharp corner.
Practice Problems
1.
State the domain and range of $y=\dfrac{1}{x}$.
Domain: $(-\infty,0)\cup(0,\infty)$; Range: same.
2.
Find $f(-1)$, $f(0)$, $f(1)$ for $f(x)=x^3-x$.
$f(-1)=0$; $f(0)=0$; $f(1)=0$.
3.
Is $f(x)=2x+5$ one-to-one? Explain.
Yes — it is a non-horizontal line; HLT passes.
4.
Is $g(x)=x^4$ one-to-one on all reals?
No — $g(1)=g(-1)=1$; fails HLT.
5.
State the domain of $h(x)=\sqrt{4-x^2}$.
$4-x^2\geq 0\Rightarrow -2\leq x\leq 2$, so $[-2,2]$.
6.
Which library function has a sharp corner at the origin and range $[0,\infty)$?
The absolute value function $y=|x|$.
