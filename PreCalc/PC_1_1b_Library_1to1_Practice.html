Library of Functions & One-to-One

PreCalc  ·  §1.1b  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Parent Functions Key parent functions: $f(x)=c$ (constant), $f(x)=x$ (identity), $f(x)=x^2$ (squaring), $f(x)=x^3$ (cubing), $f(x)=\sqrt{x}$ (square root), $f(x)=|x|$ (absolute value), $f(x)=\frac{1}{x}$ (reciprocal), $f(x)=\sqrt[3]{x}$ (cube root). Each has a characteristic shape and domain/range.
One-to-One Functions A function is one-to-one (1-to-1) if each output value is produced by exactly one input. The horizontal line test: a function is 1-to-1 if and only if every horizontal line crosses its graph at most once. Only 1-to-1 functions have inverses that are also functions.
Domain and Range of Parent Functions $f(x)=\sqrt{x}$: domain $[0,\infty)$, range $[0,\infty)$. $f(x)=\frac{1}{x}$: domain $(-\infty,0)\cup(0,\infty)$, range $(-\infty,0)\cup(0,\infty)$. $f(x)=x^2$: domain $(-\infty,\infty)$, range $[0,\infty)$.
