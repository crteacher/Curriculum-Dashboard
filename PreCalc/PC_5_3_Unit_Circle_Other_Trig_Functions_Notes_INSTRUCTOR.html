Unit Circle: Other Trig Functions

Pre-Calculus  ·  §5.3  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

Once you know sine and cosine, how do you define the other four trig functions? Which ones are undefined at $\theta=\pi/2$, and why?
Tangent, Cotangent, Secant, Cosecant

$$\tan\theta=\dfrac{\sin\theta}{\cos\theta},\quad \cot\theta=\dfrac{\cos\theta}{\sin\theta}$$

$$\sec\theta=\dfrac{1}{\cos\theta},\quad \csc\theta=\dfrac{1}{\sin\theta}$$

Undefined: $\tan,\sec$ undefined when $\cos\theta=0$; $\cot,\csc$ undefined when $\sin\theta=0$.

Reciprocal and Pythagorean Identities

Pythagorean identities:

$\sin^2\theta+\cos^2\theta=1$

$1+\tan^2\theta=\sec^2\theta$

$1+\cot^2\theta=\csc^2\theta$

Signs (ASTC): All/Sin/Tan/Cos positive in QI/II/III/IV.

Worked Examples
Example 1 — Evaluate all 6 trig functions
Find all six trig functions for $\theta=\dfrac{\pi}{3}$.
$\sin=\dfrac{\sqrt3}{2},\cos=\dfrac{1}{2},\tan=\sqrt3,\csc=\dfrac{2\sqrt3}{3},\sec=2,\cot=\dfrac{\sqrt3}{3}$.
Example 2 — Use Pythagorean identity
If $\tan\theta=2$ and $\theta$ is in QI, find $\sec\theta$.
$1+4=\sec^2\theta\Rightarrow\sec\theta=\sqrt5$.
Example 3 — Evaluate with ASTC
Find $\tan\dfrac{5\pi}{4}$ and $\sec\dfrac{5\pi}{4}$.
QIII: $\tan>0$, $\sec<0$. Reference $\dfrac{\pi}{4}$: $\tan=1$; $\sec=-\sqrt2$.
Example 4 — Find all functions from one
If $\sin\theta=-\dfrac{5}{13}$ and $\cos\theta>0$, find $\tan\theta$ and $\csc\theta$.
QIV: $\cos\theta=\dfrac{12}{13}$. $\tan=-\dfrac{5}{12}$; $\csc=-\dfrac{13}{5}$.
Practice Problems
1.
Evaluate $\tan\dfrac{\pi}{6}$.
$\dfrac{1/2}{\sqrt3/2}=\dfrac{1}{\sqrt3}=\dfrac{\sqrt3}{3}$.
2.
Find $\csc\dfrac{\pi}{2}$.
$1/\sin(\pi/2)=1/1=1$.
3.
If $\cos\theta=-\dfrac{3}{5}$ in QIII, find $\tan\theta$.
$\sin=-\tfrac{4}{5}$; $\tan=\tfrac{4}{3}$.
4.
Simplify $\sec^2\theta-1$.
$\tan^2\theta$.
5.
Find all $\theta\in[0,2\pi)$ where $\tan\theta$ is undefined.
$\theta=\dfrac{\pi}{2},\dfrac{3\pi}{2}$.
6.
Evaluate $\cot\dfrac{5\pi}{6}$.
Reference $\pi/6$ in QII: $\cot=-\sqrt3$.
