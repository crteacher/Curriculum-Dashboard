Graphing Trigonometric Functions

Pre-Calculus  ·  §6.0  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

The height of a Ferris wheel car repeats every full revolution. What kind of function repeats like that, and what information would you need to write an equation for it?
Sine and Cosine Graphs

$y=A\sin(B(x-C))+D$ or $y=A\cos(B(x-C))+D$

  • Amplitude: $|A|$ (half the height)
  • Period: $\dfrac{2\pi}{|B|}$
  • Phase shift: $C$ (right if $C>0$)
  • Vertical shift: $D$ (midline at $y=D$)

$y=\sin x$: period $2\pi$, amp 1, passes through $(0,0)$.
$y=\cos x$: period $2\pi$, amp 1, passes through $(0,1)$.

▸ Show Proof

Here's why the period is $\dfrac{2\pi}{|B|}$: the period $T$ is the smallest positive number that makes the graph repeat, meaning $\sin(B(x+T)) = \sin(Bx)$ for every $x$.

Plain sine repeats every $2\pi$ — that is, $\sin(\theta+2\pi)=\sin\theta$ for any $\theta$. So the graph repeats exactly when the *input* to sine increases by $2\pi$ as $x$ increases by $T$:

$$B(x+T) = Bx + 2\pi$$

Subtract $Bx$ from both sides: $BT = 2\pi$, so $T = \dfrac{2\pi}{B}$.

If $B$ is negative, the same repeat happens after $|B|\cdot T = 2\pi$ — hence the absolute value, $T=\dfrac{2\pi}{|B|}$. The same argument works for cosine, since $\cos$ also repeats every $2\pi$.

∎

▸ Common Mistake

Given $y=A\sin(Bx-C)+D$, students grab the number next to $x$ and call it the phase shift — reading $C$ straight off without dividing by $B$. That only works when $B=1$. The formula above is written as $B(x-C)$, already factored; if your equation comes as $Bx-C$ instead, you have to factor $B$ out first: $Bx-C=B\left(x-\dfrac{C}{B}\right)$, so the true shift is $\dfrac{C}{B}$.

This is exactly what happens in Example 1: $y=3\sin(2x-\pi)+1$ looks like it shifts $\pi$ right, but $B=2$, so the real phase shift is $\pi/2$. Skipping the factoring step is the single most common error on this topic.

Key Points and Sketching

Five key points for one period (sine):
$(0,0)\to(T/4,A)\to(T/2,0)\to(3T/4,-A)\to(T,0)$

For transformed: apply the shifts to each key point.

Midline: $y=D$; range $[D-|A|,\,D+|A|]$.

▸ Common Mistake

Students treat a negative $A$ as just "a smaller number" and still plot the first key point going up, the way plain sine does. A negative amplitude isn't a size adjustment — it flips the whole graph over the midline. For $y=A\sin(B(x-C))+D$ with $A<0$, the five key points become $(0,D)\to(T/4,D+A)\to(T/2,D)\to(3T/4,D-A)\to(T,D)$, so the curve heads down first, not up.

You can see this in Problem 1, $y=-4\cos(3x)$: it doesn't start at a maximum like ordinary cosine does — it starts at its minimum, $y=-4$, because the sign of $A$ reflects the graph before any shift is even applied.

Worked Examples
Example 1 — Identify parameters
State amp, period, phase shift, vertical shift for $y=3\sin(2x-\pi)+1$.
Write as $3\sin(2(x-\pi/2))+1$. Amp $=3$; Period $=\pi$; Phase shift $=\pi/2$ right; VS $=1$.
Example 2 — Sketch a sine graph
Sketch $y=2\sin(\pi x)$ over two periods.
Amp $=2$, Period $=\dfrac{2\pi}{\pi}=2$. Key points: $(0,0),(0.5,2),(1,0),(1.5,-2),(2,0)$.
Example 3 — Write equation from graph
A graph has amplitude 4, period $\pi$, midline $y=2$, starts at a maximum. Write its equation.
$y=4\cos\left(\dfrac{2\pi}{\pi}x\right)+2=4\cos(2x)+2$.
Example 4 — Ferris wheel model
A Ferris wheel has diameter 40 m, center 25 m up, rotates once in 60 s. Write $h(t)$.
$h(t)=-20\cos\left(\dfrac{2\pi}{60}t\right)+25=-20\cos\left(\dfrac{\pi}{30}t\right)+25$.
Practice Problems
1.
State amplitude and period of $y=-4\cos(3x)$.
Amp $=4$; Period $=\dfrac{2\pi}{3}$.
2.
What is the range of $y=5\sin(x)+2$?
$[-3,7]$.
3.
Find phase shift of $y=\sin\left(x-\dfrac{\pi}{4}\right)$.
Right $\dfrac{\pi}{4}$.
4.
Write equation: amplitude 2, period 4, no shift, cosine.
$y=2\cos\left(\dfrac{\pi}{2}x\right)$.
5.
Give the midline of $y=3\cos(x)-5$.
$y=-5$.
6.
A tide model: $h(t)=6\sin\left(\dfrac{\pi}{6}t\right)+10$. What is the max tide height?
Amp $=6$, midline $=10$: max $=16$ m.
