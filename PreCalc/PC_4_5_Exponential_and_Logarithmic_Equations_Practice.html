Exponential & Logarithmic Equations

PreCalc  ·  §4.5  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Solving exponential equations Strategy 1 — Same base: if both sides can be written as $b^{(\cdot)}$, set exponents equal.
$b^{f(x)} = b^{g(x)} \Rightarrow f(x) = g(x)$
Strategy 2 — Take logarithm: apply $\ln$ (or $\log$) to both sides.
$b^x = c \Rightarrow x \ln b = \ln c \Rightarrow x = \dfrac{\ln c}{\ln b}$
Solving logarithmic equations Strategy 1 — Condense then convert: write as a single log, then rewrite in exponential form.
$\log_b(f(x)) = c \Rightarrow f(x) = b^c$
Strategy 2 — Equal logs: $\log_b A = \log_b B \Rightarrow A = B$.
Always check: the argument of every log must be $> 0$ in the original equation (reject extraneous solutions).
