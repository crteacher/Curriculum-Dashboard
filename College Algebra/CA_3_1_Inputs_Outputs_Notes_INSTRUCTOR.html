Inputs and Outputs

College Algebra  ·  §3.1  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

Think of a function like a vending machine: you put in a code (input) and get out a snack (output). If $f(x) = 13$, what is the question actually asking — are you finding the output or the input?
Evaluating: Input → Output

$f(x)$ means: substitute the value for every $x$.

e.g. $f(x) = 3x + 4$, find $f(5)$:
$f(5) = 3(5) + 4 = 19$

e.g. $g(x) = x^2 - 2x + 1$, find $g(4)$:
$g(4) = 16 - 8 + 1 = 9$

Tip: replace every $x$ with the input, then simplify using order of operations.

Reversing: Output → Input

Given $f(x) = k$, set the formula equal to $k$ and solve for $x$.

e.g. $f(x) = 4x - 3$, if $f(x) = 17$, find $x$:
$4x - 3 = 17 \Rightarrow 4x = 20 \Rightarrow x = 5$

Tip: reversing means solving an equation — isolate $x$, just like in §1.1.

Worked Examples
Example 1  —  Evaluate a linear function
Let $f(x) = 3x + 4$. Find $f(5)$.
$f(5) = 3(5) + 4 = 15 + 4 = 19$
Example 2  —  Evaluate a quadratic function
Let $g(x) = x^2 - 2x + 1$. Find $g(4)$.
$g(4) = 16 - 8 + 1 = 9$
Example 3  —  Reverse: find the input
Let $f(x) = 4x - 3$. If $f(x) = 17$, find $x$.
$4x - 3 = 17 \Rightarrow 4x = 20 \Rightarrow x = 5$
Example 4  —  Evaluate with a negative input
Let $h(x) = 2x^2 + x$. Find $h(-3)$.
$h(-3) = 2(-3)^2 + (-3) = 2(9) - 3 = 18 - 3 = 15$
Practice Problems
1.
Let $f(x) = 5x - 2$. Find $f(3)$.
$f(3) = 13$
2.
Let $g(x) = x^2 + 1$. Find $g(-2)$.
$g(-2) = 4 + 1 = 5$
3.
Let $f(x) = 3x + 9$. If $f(x) = 0$, find $x$.
$3x+9=0 \Rightarrow x=-3$
4.
Let $h(x) = 2x - 7$. Find $h(0)$.
$h(0) = -7$
5.
Let $f(x) = x^2 - 4$. Find $f(-3)$.
$f(-3) = 9 - 4 = 5$
6.
Let $f(x) = -2x + 10$. If $f(x) = 4$, find $x$.
$-2x+10=4 \Rightarrow x=3$
