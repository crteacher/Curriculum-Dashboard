Graphing Exponential Functions

College Algebra  ·  §6.4  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
y-Intercept For $y = a\cdot b^x$, the graph always crosses the $y$-axis at $(0,\,a)$ because $b^0 = 1$.
Multiply-by-b Pattern Each time $x$ increases by 1, multiply $y$ by $b$. Growth ($b>1$): curve rises. Decay ($0 < b < 1$): curve falls.
Horizontal Asymptote The $x$-axis ($y=0$) is a horizontal asymptote — the curve approaches but never crosses it. There is no $x$-intercept (for $a>0$).
