Domain and Range

College Algebra  ·  §3.2  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

A square root can't take a negative number, and a fraction can't have 0 in the denominator. What two things should you always watch for when finding the domain of a function?
Domain

Domain = all valid inputs (x-values).

Danger zones to exclude:

  • Denominator = 0: set denominator $\neq 0$
  • Even root of negative: set expression $\geq 0$

e.g. $f(x) = \dfrac{3x}{x-2}$: exclude $x = 2$
Domain: $(-\infty, 2) \cup (2, \infty)$

e.g. $h(x) = \sqrt{x-3}$: need $x - 3 \geq 0$
Domain: $[3, \infty)$

Range

Range = all possible outputs (y-values).

Quick shortcuts by function type:

  • Linear $y = mx+b$: range $(-\infty, \infty)$
  • Quadratic $y = x^2 + k$: range $[k, \infty)$ (opens up) or $(-\infty, k]$ (opens down)
  • Square root $y = \sqrt{x}$: range $[0, \infty)$

e.g. $f(x) = x^2 + 4$:
Minimum value is 4 → range $[4, \infty)$

Worked Examples
Example 1  —  Polynomial domain
State the domain of $f(x) = x^2 + 5x$.
No restrictions → domain $(-\infty, \infty)$
Example 2  —  Rational function domain
State the domain of $g(x) = \dfrac{3x}{x-2}$.
$x - 2 \neq 0 \Rightarrow x \neq 2$
Domain: $(-\infty, 2) \cup (2, \infty)$
Example 3  —  Square root domain
State the domain of $h(x) = \sqrt{x-3}$.
$x - 3 \geq 0 \Rightarrow x \geq 3$
Domain: $[3, \infty)$
Example 4  —  Range of a quadratic
State the range of $f(x) = x^2 + 4$.
Minimum at $x=0$: $f(0)=4$. Range: $[4, \infty)$
Practice Problems
1.
State the domain of $f(x) = 3x - 7$.
$(-\infty, \infty)$
2.
State the domain of $g(x) = \dfrac{x+1}{x+4}$.
$x \neq -4$; $(-\infty,-4) \cup (-4,\infty)$
3.
State the domain of $h(x) = \sqrt{2x-6}$.
$2x \geq 6 \Rightarrow x \geq 3$; $[3, \infty)$
4.
State the range of $f(x) = -x^2 + 9$.
Maximum at $x=0$: $f(0)=9$. Range: $(-\infty, 9]$
5.
State the domain of $f(x) = \dfrac{5}{x^2-1}$.
$x \neq \pm 1$; $(-\infty,-1) \cup (-1,1) \cup (1,\infty)$
6.
State the range of $y = \sqrt{x}$.
$[0, \infty)$
