Inverse Functions

College Algebra  ·  §3.5  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

An inverse function undoes what the original function does. What property must $f$ have so that for each output, there is exactly one input that produced it? Why would a parabola fail this test?
One-to-One & the Horizontal Line Test

One-to-one function: each output comes from exactly one input.

Horizontal Line Test: a function has an inverse if and only if every horizontal line crosses its graph at most once.

e.g. $f(x) = 3x - 9$: a line → passes HLT → one-to-one

e.g. $f(x) = x^2 - 4$: parabola → fails HLT → not one-to-one

Notation: $f^{-1}(x)$ denotes the inverse function.

Finding the Inverse Algebraically

Steps:

  1. Replace $f(x)$ with $y$
  2. Swap $x$ and $y$
  3. Solve for $y$
  4. Write as $f^{-1}(x)$

Key property: $f^{-1}(f(x)) = x$ and $f(f^{-1}(x)) = x$

e.g. $f(x) = 3x - 9$: swap → $x = 3y - 9$ → $y = \dfrac{x+9}{3}$

Worked Examples
Example 1  —  Find the inverse
Find $f^{-1}(x)$ for $f(x) = 3x - 9$.
$y = 3x-9 \Rightarrow x=3y-9 \Rightarrow y = \dfrac{x+9}{3}$
$f^{-1}(x) = \dfrac{x+9}{3}$
Example 2  —  Inverse composition
Given $f(x) = 5x + 2$, find $f^{-1}(f(7))$.
$f^{-1}(f(7)) = 7$   (inverse undoes the function)
Example 3  —  Evaluate the inverse
Given $f(x) = 4x - 8$, find $f^{-1}(12)$.
$f^{-1}(x) = \dfrac{x+8}{4}$;   $f^{-1}(12) = \dfrac{20}{4} = 5$
Example 4  —  Horizontal line test
Is $f(x) = x^2 - 4$ one-to-one? Does it have an inverse?
Fails the horizontal line test → not one-to-one, no inverse over all reals
Practice Problems
1.
Find $f^{-1}(x)$ for $f(x) = 2x + 6$.
$f^{-1}(x) = \dfrac{x-6}{2}$
2.
Find $f^{-1}(x)$ for $f(x) = \dfrac{x}{3} - 1$.
$f^{-1}(x) = 3(x+1) = 3x+3$
3.
Given $f(x) = 7x - 3$, find $f^{-1}(f(4))$.
$f^{-1}(f(4)) = 4$
4.
Does $f(x) = x^3$ have an inverse?
Yes — passes the horizontal line test
5.
Find $f^{-1}(10)$ given $f(x) = 5x$.
$f^{-1}(x) = \dfrac{x}{5}$; $f^{-1}(10)=2$
6.
Is $f(x) = |x|$ one-to-one?
No — fails HLT ($f(2)=f(-2)=2$)
