Inverse Functions

College Algebra  ·  §3.5  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

An inverse function undoes what the original function does. What property must $f$ have so that for each output, there is exactly one input that produced it? Why would a parabola fail this test?
One-to-One & the Horizontal Line Test

One-to-one function: each output comes from exactly one input.

Horizontal Line Test: a function has an inverse if and only if every horizontal line crosses its graph at most once.

e.g. $f(x) = 3x - 9$: a line → passes HLT → one-to-one

e.g. $f(x) = x^2 - 4$: parabola → fails HLT → not one-to-one

Notation: $f^{-1}(x)$ denotes the inverse function.

▸ Show Proof

An inverse only makes sense if reversing the function is unambiguous — given an output, there has to be exactly one input to send it back to. That's precisely what "one-to-one" means, so we just need to connect one-to-one to the horizontal line test.

A horizontal line $y=k$ crosses the graph once for every input $x$ with $f(x)=k$. So the number of times that line crosses the graph is exactly the number of inputs that produce the output $k$.

If some horizontal line crosses the graph twice, that means two different inputs $x_1\neq x_2$ both satisfy $f(x_1)=f(x_2)=k$ — the same output came from two different inputs. Trying to reverse this would leave you asking "does $f^{-1}(k)$ equal $x_1$ or $x_2$?" with no way to decide. No inverse can exist.

If instead every horizontal line crosses at most once, every output comes from exactly one input — the function is one-to-one, and the inverse is well-defined by simply reversing each pair.

∎

▸ Common Mistake

Students read $f^{-1}(x)$ as "$f$ to the negative one power" and write $f^{-1}(x)=\dfrac{1}{f(x)}$. It isn't — the $-1$ here is inverse-function notation, not an exponent. For $f(x)=3x-9$, $\dfrac{1}{f(x)}=\dfrac{1}{3x-9}$, but $f^{-1}(x)=\dfrac{x+9}{3}$ — completely different functions.

The confusion comes from the fact that $x^{-1}$ really does mean $\dfrac{1}{x}$ for numbers, so the notation looks inconsistent — but function composition doesn't follow the same exponent rules as multiplication. You can check which one you've built by testing $f^{-1}(f(x))=x$; the reciprocal $\dfrac{1}{f(x)}$ will never satisfy that.
