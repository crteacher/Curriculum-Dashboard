Core Ideas
Substitution / Elimination Solve a 2×2 system of linear equations by eliminating one variable — multiply one or both equations so a variable cancels when added, or solve one equation for a variable and substitute it into the other.
Check Your Answer Plug both values back into both original equations to confirm the solution works.
