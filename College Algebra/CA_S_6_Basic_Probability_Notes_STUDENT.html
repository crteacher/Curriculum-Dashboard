Basic Probability

College Algebra  ·  §S.6  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

A fair die has 6 equally likely outcomes. The probability formula $P(A) = \dfrac{|A|}{|S|}$ only works when all outcomes are equally likely. What must be true about the sample space for this fraction to be valid?
The Basic Probability Formula

$$P(A) = \frac{|A|}{|S|}$$

$A$ = event (set of favorable outcomes)
$S$ = sample space (set of all possible outcomes)

Properties:
$0 \leq P(A) \leq 1$
$P(S) = 1$    $P(\emptyset) = 0$

e.g. Roll a die; $P(\text{rolling} > 4) = \dfrac{2}{6} = \dfrac{1}{3}$

Complement and Mutually Exclusive

Complement rule:
$P(A') = 1 - P(A)$

e.g. $P(\text{not rolling a 3}) = 1 - \dfrac{1}{6} = \dfrac{5}{6}$

Mutually exclusive events:
$A \cap B = \emptyset$, so $P(A \cup B) = P(A) + P(B)$

e.g. $P(\text{rolling 1 or 2}) = \dfrac{1}{6} + \dfrac{1}{6} = \dfrac{2}{6} = \dfrac{1}{3}$

Worked Examples
Example 1  —  Basic probability
Roll a fair die. Find $P(\text{rolling} > 4)$.
Favorable: $\{5,6\}$; $P = \dfrac{2}{6} = \dfrac{1}{3}$
Example 2  —  Complement rule
Roll a fair die. Find $P(\text{not rolling a 3})$.
$P = 1 - \dfrac{1}{6} = \dfrac{5}{6}$
Example 3  —  Card probability
Draw one card from a standard 52-card deck. Find $P(\text{drawing a King})$.
$P = \dfrac{4}{52} = \dfrac{1}{13}$
Example 4  —  Mutually exclusive events
Roll a fair die. Find $P(\text{rolling 1 or 2})$.
$P = \dfrac{1}{6} + \dfrac{1}{6} = \dfrac{2}{6} = \dfrac{1}{3}$
Practice Problems
1.
Roll a fair die. Find $P(\text{rolling an even number})$.
$\dfrac{3}{6} = \dfrac{1}{2}$
2.
Roll a fair die. Find $P(\text{not rolling a 1 or 2})$.
$1 - \dfrac{2}{6} = \dfrac{4}{6} = \dfrac{2}{3}$
3.
Draw a card. Find $P(\text{drawing an Ace})$.
$\dfrac{4}{52} = \dfrac{1}{13}$
4.
A bag has 3 red and 7 blue marbles. Find $P(\text{red})$.
$\dfrac{3}{10}$
5.
$P(A) = 0.4$. Find $P(A')$.
$1 - 0.4 = 0.6$
6.
A die is rolled. Find $P(\text{rolling} \leq 2)$.
$\dfrac{2}{6} = \dfrac{1}{3}$
