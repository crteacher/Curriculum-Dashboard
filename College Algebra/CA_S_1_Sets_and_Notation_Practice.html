Sets and Notation

College Algebra  ·  §S.1  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Set Membership $a \in A$ means $a$ is an element of set $A$.   $a \notin A$ means it is not.   A set is defined by its members — order and repeats do not matter.
Cardinality $|A|$ (or $n(A)$) is the number of distinct elements in set $A$.   $|\emptyset| = 0$.   Repeated elements count only once.
Notation Roster: list elements in braces — $\{1,2,3\}$.   Set builder: $\{n \mid \text{condition}\}$, read "the set of all $n$ such that …"
