Solutions to Linear Equations

College Algebra  ·  §1.2  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

An ordered pair (x, y) is a solution to an equation if substituting those values makes the equation true. If (3, 11) lies on the graph of f(x) = ax + 2, what must a equal?
Function Evaluation

Evaluating f(x):
Substitute the input value for every x.
e.g. $f(x) = 4x + 3$, find $f(2)$:
$f(2) = 4(2) + 3 = 11$

Checking a solution (x, y):
Substitute x into f(x). If the output equals y, then the ordered pair is a solution.

Strategy: replace x with the given input, simplify completely, and compare to the expected output.

Finding Solutions & Intersection

Complete an ordered pair:
Given (x, __), substitute x to find y.
e.g. $y = 3x - 1$, complete $(4, \text{__})$:
$y = 3(4) - 1 = 11 \Rightarrow (4, 11)$

Intersection of two lines:
Set $f(x) = g(x)$ and solve for x.
e.g. $f(x) = 3x + 1$, $g(x) = x + 7$:
$3x + 1 = x + 7 \Rightarrow x = 3$

Worked Examples
Example 1  —  Evaluate a function
Let $f(x) = 3x + 5$. Find $f(4)$.
$f(4) = 3(4) + 5 = 17$
Example 2  —  Check a solution
Is $(3, 8)$ a solution to $y = 2x + 2$?
$2(3) + 2 = 8$ ✓ — Yes
Example 3  —  Complete an ordered pair
Complete the ordered pair $(4, \text{__})$ for $y = 3x - 1$.
$y = 3(4) - 1 = 11 \Rightarrow (4, 11)$
Example 4  —  Intersection of two lines
Find x where $f(x) = 3x + 1$ and $g(x) = x + 7$ intersect.
$3x + 1 = x + 7 \Rightarrow 2x = 6 \Rightarrow x = 3$
Practice Problems
1.
Let $f(x) = 2x - 3$. Find $f(5)$.
$f(5) = 7$
2.
Is $(2, 7)$ a solution to $y = 3x + 1$?
Yes ($3(2)+1=7$)
3.
Complete $(3, \text{__})$ for $y = 4x - 2$.
$y = 10$; ordered pair is $(3, 10)$
4.
Is $(-1, 5)$ a solution to $y = -2x + 3$?
No ($-2(-1)+3=5$, actually yes — Yes)
5.
Find x where $f(x) = 2x + 3$ and $g(x) = 5x - 6$ are equal.
$2x+3=5x-6 \Rightarrow x=3$
6.
Complete $(0, \text{__})$ for $y = -3x + 7$.
$y = 7$; ordered pair is $(0, 7)$
