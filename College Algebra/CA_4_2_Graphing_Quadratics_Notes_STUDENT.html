Graphing Quadratics

College Algebra  ·  §4.2  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

A football thrown in the air, an arch on a bridge, the cables of a suspension bridge — all parabolas. The equation $f(x)=ax^2+bx+c$ controls everything: which way it opens, how wide, and where the turning point sits. Before graphing anything, what single number in the equation tells you whether the parabola opens up or down?
Standard Form $f(x)=ax^2+bx+c$

$a>0$ → opens up (vertex is a minimum)
$a<0$ → opens down (vertex is a maximum)

Axis of symmetry: $x=-\dfrac{b}{2a}$

Vertex:
  $x$-coordinate: $h=-\dfrac{b}{2a}$
  $y$-coordinate: $k=f(h)$

$y$-intercept: always at $(0,c)$

Example: $f(x)=x^2+4x+1$:
$h=-4/2=-2$, $k=4-8+1=-3$
Vertex: $(-2,-3)$   Axis: $x=-2$

Vertex Form $f(x)=a(x-h)^2+k$

Vertex $(h,k)$ is read directly from the equation.

$f(x)=2(x-3)^2+1$ → vertex $(3,1)$, opens up

$f(x)=-(x+2)^2-4$: since $x+2=x-(-2)$, vertex $(-2,-4)$, opens down

Finding the $y$-intercept from vertex form:
Set $x=0$: $f(0)=ah^2+k$

Example: $f(x)=(x-3)^2-1$:
$f(0)=9-1=8$ → $y$-intercept $(0,8)$

Worked Examples
Example 1  —  Vertex from standard form
Find the vertex of $f(x)=x^2-6x+5$.
$h=\dfrac{6}{2}=3$,   $k=9-18+5=-4$
Vertex: $(3,-4)$
Example 2  —  Vertex from vertex form
State vertex and direction for $g(x)=-2(x+1)^2+5$.
$h=-1$, $k=5$, $a=-2<0$ → opens down
Vertex: $(-1,5)$ — maximum
Example 3  —  $y$-intercept
Find the $y$-intercept of $f(x)=x^2+2x-8$.
Set $x=0$: $f(0)=-8$
$y$-intercept: $(0,-8)$
Example 4  —  Axis of symmetry
Find the axis of symmetry of $f(x)=2x^2+8x+3$.
$x=-\dfrac{8}{4}=-2$   Axis: $x=-2$
Practice Problems
1.
$f(x)=x^2+6x+4$. Find the $x$-coordinate of the vertex.
$h=-3$
2.
$f(x)=-(x-2)^2+7$. State the vertex and direction.
$(2,7)$, opens down
3.
$f(x)=x^2-4x+1$. Find the vertex.
$h=2$, $k=-3$ → $(2,-3)$
4.
$f(x)=3x^2-12x+5$. Find the axis of symmetry.
$x=-(-12)/(6)=2$
5.
$f(x)=x^2-2x-8$. Find the $y$-intercept.
$(0,-8)$
6.
$f(x)=-x^2+4x+1$. Does the parabola open up or down? Find the vertex.
Opens down; $h=2$, $k=5$ → vertex $(2,5)$
