Set Operations

College Algebra  ·  §S.2  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Union & Intersection $A \cup B$: elements in $A$ OR $B$ (combine all).   $A \cap B$: elements in $A$ AND $B$ (keep only the overlap).   Disjoint sets have $A \cap B = \emptyset$.
Complement $A'$ (or $A^c$): all elements of the universal set $U$ that are NOT in $A$.   Always defined relative to $U$.
Set Subtraction $A \setminus B$: elements of $A$ that are not in $B$.   Equivalent to $A \cap B'$.   Order matters: $A \setminus B \neq B \setminus A$ in general.
