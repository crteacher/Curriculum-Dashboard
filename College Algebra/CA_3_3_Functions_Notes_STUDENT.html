Functions

College Algebra  ·  §3.3  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

A relation is any pairing of inputs with outputs. But a function has one extra rule: each input is paired with exactly one output. Your student ID maps to exactly one person — that’s a function. A single person might have multiple phone numbers — that would NOT be a function in reverse. What test on a graph tells you instantly whether a relation is a function?
The Function Rule

A function: each input ($x$) maps to exactly one output ($y$).

Testing ordered pairs or a table: check whether any $x$-value appears more than once with a different $y$-value.

  • $(1,3),(2,5),(3,5)$ → function (same $y$ from different $x$ is fine)
  • $(1,3),(1,7),(2,4)$ → not a function ($x=1$ has two outputs)
The Vertical Line Test

From a graph: if any vertical line hits the graph more than once → not a function (one $x$, two $y$-values).

If every vertical line hits at most once → function.

  • $y=2x+1$ ✓   $y=x^2$ ✓
  • $x=y^2$ ✗   circle ✗

Note: $x=a$ (vertical line) is not a function. $y=b$ (horizontal line) is a function.

Worked Examples
Example 1  —  Ordered pairs — function?
$\{(1,4),(2,7),(3,4),(4,9)\}$
No $x$-value repeats. Each $x$ has exactly one $y$.
Yes — function. (Note: $y=4$ appears twice but for different $x$.)
Example 2  —  Table — function?
$x$: 1, 2, 3, 2   $y$: 5, 8, 11, 3
$x=2$ maps to both 8 and 3.
Not a function.
Example 3  —  Equation — function?
Is $y=3x-2$ a function?
For every $x$ there is exactly one $y$.
Yes — function.
Example 4  —  Equation — function?
Is $x=y^2$ a function?
$x=4$ gives $y=2$ or $y=-2$.
One input, two outputs → Not a function.
Practice Problems
1.
$\{(2,5),(3,8),(4,5),(5,10)\}$ — function?
Yes — no $x$ repeats
2.
$\{(1,3),(2,6),(1,9),(3,12)\}$ — function?
No — $x=1$ maps to 3 and 9
3.
$x$: 1,2,3,4   $y$: 4,4,4,4 — function?
Yes — each $x$ has exactly one $y$ (all equal 4)
4.
Is $y=x^2+1$ a function?
Yes — one output per input
5.
Is $x^2+y^2=9$ a function?
No — fails vertical line test (e.g. $x=0$ gives $y=\pm 3$)
6.
Is $y=5$ a function?
Yes — every $x$ maps to exactly one output (5)
