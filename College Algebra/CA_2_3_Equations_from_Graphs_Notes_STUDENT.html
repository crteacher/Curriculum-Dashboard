Equations from Graphs

College Algebra  ·  §2.3  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

What two pieces of information do you need to write the equation of a line? If you have a graph, how do you read off those two pieces?
When the y-Intercept Is Visible

Step 1: Read the y-intercept $(0, b)$ from the graph.

Step 2: Count rise over run between two clear points to find slope $m$.

Step 3: Write $y = mx + b$.

Example: Graph crosses y-axis at 2, passes through $(3, 8)$:
$m = \dfrac{8-2}{3-0} = 2$, so $y = 2x + 2$

Count grid squares carefully: rise is vertical change, run is horizontal change. Rise/run can be negative if the line falls.

When y-Intercept Is Not Obvious

Use two identifiable points and point-slope form.

e.g. through $(1, 7)$ and $(4, 1)$:
$m = \dfrac{1-7}{4-1} = -2$
$y - 7 = -2(x-1) \Rightarrow y = -2x + 9$

Special cases:
Horizontal line through $(3, -4)$: $y = -4$
Vertical line through $(5, 2)$: $x = 5$

Worked Examples
Example 1  —  y-intercept visible
A line crosses the y-axis at 2 and passes through $(3, 8)$. Write its equation.
$m = \frac{8-2}{3-0} = 2$;   $y = 2x + 2$
Example 2  —  Two points given
Write the equation of the line through $(1, 7)$ and $(4, 1)$.
$m = \frac{1-7}{4-1} = -2$;   $y-7=-2(x-1) \Rightarrow y = -2x + 9$
Example 3  —  Horizontal line
Write the equation of the horizontal line through $(3, -4)$.
$y = -4$
Example 4  —  Vertical line
Write the equation of the vertical line through $(5, 2)$.
$x = 5$
Practice Problems
1.
A line crosses the y-axis at $-3$ and passes through $(2, 5)$. Write its equation.
$m=\frac{5-(-3)}{2-0}=4$; $y=4x-3$
2.
Write the equation through $(0, -1)$ and $(3, 5)$.
$m=2$; $y=2x-1$
3.
Write the equation through $(2, 3)$ and $(6, 7)$.
$m=1$; $y=x+1$
4.
Write the equation of the horizontal line through $(-2, 6)$.
$y=6$
5.
Write the equation of the vertical line through $(4, -1)$.
$x=4$
6.
Write the equation through $(-1, 5)$ and $(3, -3)$.
$m=-2$; $y=-2x+3$
