Absolute Values

College Algebra  ·  §1.6  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

$|x| = k$ has two solutions when $k > 0$, exactly one when $k = 0$, and no solution when $k < 0$. Why does a negative right-hand side always produce no solution?
Absolute Value Equations

Definition: $|x|$ = distance from 0 on the number line.

To solve $|\text{expr}| = k$:
If $k > 0$: set expr $= k$ or expr $= -k$, solve both.
If $k = 0$: set expr $= 0$, one solution.
If $k < 0$: no solution (distance can't be negative).

Important: isolate the absolute value first before splitting into two cases.

▸ Common Mistake

Isolate the absolute value before splitting into two cases. On $2|x+1|-2=5$, students jump straight to $x+1=5$ or $x+1=-5$ — but $|x+1|$ isn't alone yet.

Solve for it first: $|x+1|=\frac72$, then split. Splitting too early uses the wrong number on the right and produces wrong solutions.

Strategy: isolate $|\ldots|$ → check sign of RHS → split into $+$ and $-$ cases.

Absolute Value Inequalities

Less than (between):
$|\text{expr}| < k \Rightarrow -k < \text{expr} < k$
Solution is a single interval.

Greater than (outside):
$|\text{expr}| > k \Rightarrow \text{expr} < -k$ OR $\text{expr} > k$
Solution is a union of two intervals.

Remember: $\leq$ and $\geq$ follow the same pattern with closed brackets where needed.

▸ Common Mistake

Less-than is AND (between); greater-than is OR (outside) — don't swap them. $|\text{expr}|k$ becomes two pieces: $\text{expr}<-k$ OR $\text{expr}>k$.

Writing $|\text{expr}|>k$ as $-k<\text{expr}and" (AND), "great-or" (OR).

Worked Examples
Example 1  —  Basic absolute value equation
Solve $|x - 3| = 5$.
$x - 3 = 5 \Rightarrow x = 8$   or   $x - 3 = -5 \Rightarrow x = -2$
Solution: $\{-2, 8\}$
Example 2  —  Isolate first
Solve $2|x + 1| - 2 = 5$.
$2|x+1| = 7 \Rightarrow |x+1| = \frac{7}{2}$
$x + 1 = \frac{7}{2} \Rightarrow x = \frac{5}{2}$   or   $x + 1 = -\frac{7}{2} \Rightarrow x = -\frac{9}{2}$
Example 3  —  Absolute value inequality (less than)
Solve $|2x - 4| < 6$ and write in interval notation.
$-6 < 2x - 4 < 6 \Rightarrow -2 < 2x < 10 \Rightarrow -1 < x < 5 \Rightarrow (-1, 5)$
Example 4  —  Absolute value inequality (greater than)
Solve $|2x - 4| > 6$ and write in interval notation.
$2x - 4 < -6$ or $2x - 4 > 6$
$x < -1$ or $x > 5 \Rightarrow (-\infty, -1) \cup (5, \infty)$
