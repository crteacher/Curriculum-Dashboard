Subsets and Supersets

College Algebra  ·  §S.3  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Subset $A \subseteq B$: every element of $A$ is also in $B$.   $A \subset B$ (proper subset): $A \subseteq B$ and $A \neq B$.   $\emptyset \subseteq A$ for any set $A$.
Counting Subsets A set with $n$ elements has exactly $2^n$ subsets.   Each element is either included or excluded — 2 choices per element.
Transitivity If $A \subseteq B$ and $B \subseteq C$, then $A \subseteq C$.   Also: $A \subseteq B$ and $B \subseteq A$ implies $A = B$.
