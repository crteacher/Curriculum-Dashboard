Sets, Counting & Probability — Unit Review

College Algebra  ·  Unit Review (§S.1–S.11)  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Sets & Notation $|A|$ (or $n(A)$) is the number of distinct elements in set $A$.   $|\emptyset| = 0$.   Repeated elements count only once.
Set Operations $A \cup B$: elements in $A$ OR $B$ (combine all).   $A \cap B$: elements in $A$ AND $B$ (keep only the overlap).   Disjoint sets have $A \cap B = \emptyset$.
Subsets A set with $n$ elements has exactly $2^n$ subsets.   Each element is either included or excluded — 2 choices per element.
Venn Diagrams $|A|$ = (only $A$) $+$ $|A\cap B|$.   $|A\cup B|$ = only $A$ $+$ $A\cap B$ $+$ only $B$.   $|U|$ = all four regions.
Inclusion-Exclusion $|A \cup B| = |A| + |B| - |A \cap B|$.   Adding both sets double-counts the overlap — subtract it once to correct.
Basic Probability For equally likely outcomes: $P(A) = \dfrac{|A|}{|S|}$ where $A$ is the favorable set and $S$ is the sample space.   Always $0 \le P(A) \le 1$.
Multiplication Principle If task 1 has $n_1$ options and task 2 has $n_2$ options (independently), the total number of outcomes is $n_1 \times n_2$. Extends to any number of consecutive choices.
Permutations & Combinations $P(n,r) = \dfrac{n!}{(n-r)!}$ when order matters.   $C(n,r) = \dfrac{n!}{r!\,(n-r)!} = \dfrac{P(n,r)}{r!}$ when order doesn't matter.
Further Counting Exactly $k$ heads in $n$ flips: choose which $k$ of the $n$ slots are heads, $C(n,k)$.   Order of selection doesn't matter — only which slots are chosen.
Partitions Split $n$ objects into labeled groups of sizes $n_1, n_2, \ldots$: $\dfrac{n!}{n_1!\,n_2!\cdots} = C(n,n_1)\times C(n-n_1,n_2)\times\cdots$
Combined Probability $P(\text{event}) = \dfrac{\text{favorable outcomes}}{\text{total outcomes}}$. Use $C(n,r)$ or $P(n,r)$ to count each, depending on whether order matters.
