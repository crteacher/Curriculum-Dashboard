Solving Equations with One Unknown

College Algebra  ·  §1.1  ·  Instructor: Christopher Halverson  ·  Fall 2026

Core Ideas
Solving Strategy Distribute → combine like terms → collect $x$-terms on one side → constants on the other → divide. Every operation must be applied to both sides.
Number of Solutions Variable remains → one solution. Variable cancels, true statement → infinite solutions. Variable cancels, false statement → no solution.
