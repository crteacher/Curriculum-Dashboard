Arc Length

Calculus 2  ·  §9  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Arc Length Formula For $y=f(x)$ on $[a,b]$: $\displaystyle L=\int_a^b\sqrt{1+[f'(x)]^2}\,dx$. Derived from $ds=\sqrt{dx^2+dy^2}=\sqrt{1+(dy/dx)^2}\,dx$ — summing tiny hypotenuses along the curve.
Arc Length with Respect to y For $x=g(y)$ on $[c,d]$: $\displaystyle L=\int_c^d\sqrt{1+[g'(y)]^2}\,dy$. Use this form when the curve is more naturally described as a function of $y$.
Sanity Check & Key Fact For a line $y=mx+b$: $f'=m$, so $L=\sqrt{1+m^2}\cdot(b-a)$ — exactly the distance formula. ✓ Many arc-length integrals have no closed form; set up the integral and approximate numerically.
