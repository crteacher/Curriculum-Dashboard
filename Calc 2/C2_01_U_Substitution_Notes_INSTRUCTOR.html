U-Substitution

Calc 2  ·  §1  ·  Christopher Halverson  ·  Fall 2026  ·  INSTRUCTOR

The Chain Rule tells us how to differentiate a composition $f(g(x))$. If differentiation uses the chain rule, what technique do you think lets us integrate compositions?
The Idea

Reverse Chain Rule:
If $F'=f$, then $\int f(g(x))\,g'(x)\,dx = F(g(x))+C$

Substitution: Let $u=g(x)$, so $du=g'(x)\,dx$
$$\int f(g(x))\,g'(x)\,dx = \int f(u)\,du$$

▸ Show Proof

Here's the key idea: substitution is just the Chain Rule read backwards. Suppose $F$ is an antiderivative of $f$, so that $F'=f$.

Differentiate the composition $F(g(x))$ using the Chain Rule:

$$\frac{d}{dx}\,F(g(x)) = F'(g(x))\,g'(x) = f(g(x))\,g'(x).$$

This says $F(g(x))$ is an antiderivative of $f(g(x))\,g'(x)$, so $\displaystyle\int f(g(x))\,g'(x)\,dx = F(g(x))+C$.

Now let $u=g(x)$. Then $\displaystyle\int f(u)\,du = F(u)+C = F(g(x))+C$ too. Both sides equal the same antiderivative, so replacing $g(x)$ by $u$ and $g'(x)\,dx$ by $du$ is exactly justified.

∎

Look for a function and its derivative both present in the integrand.

Process

1. Choose $u$ = inner function (or a piece whose derivative is also present)

2. Compute $du = u'\,dx$, solve for $dx$ if needed

3. Rewrite entirely in terms of $u$

4. Integrate in $u$

5. Back-substitute $u = g(x)$

Definite integrals: change the limits to $u$-values or convert back before evaluating.

Worked Examples
Example 1  —  Basic power
Evaluate $\int 2x(x^2+1)^4\,dx$
Let $u=x^2+1$, $du=2x\,dx$.
$=\int u^4\,du=\dfrac{u^5}{5}+C=\dfrac{(x^2+1)^5}{5}+C$
Example 2  —  Trig integrand
Evaluate $\int \cos(3x)\,dx$
Let $u=3x$, $du=3\,dx$, so $dx=\frac{du}{3}$.
$=\frac{1}{3}\int\cos u\,du=\frac{\sin(3x)}{3}+C$
Example 3  —  Exponential
Evaluate $\int x\,e^{x^2}\,dx$
Let $u=x^2$, $du=2x\,dx$.
$=\frac{1}{2}\int e^u\,du=\frac{e^{x^2}}{2}+C$
Example 4  —  Definite integral
Evaluate $\int_0^1 2x\sqrt{x^2+1}\,dx$
Let $u=x^2+1$; when $x=0$, $u=1$; when $x=1$, $u=2$.
$=\int_1^2 \sqrt{u}\,du=\left[\frac{2}{3}u^{3/2}\right]_1^2=\frac{2}{3}(2\sqrt{2}-1)$
