Hyperbolic Functions

Calculus 2  ·  §13  ·  Christopher Halverson  ·  Fall 2026

Core Ideas
Definitions & Pythagorean Identity $\sinh x = \dfrac{e^x - e^{-x}}{2}$, $\quad \cosh x = \dfrac{e^x + e^{-x}}{2}$. Key identity: $\cosh^2 x - \sinh^2 x = 1$. Note: $\cosh$ is even (U-shaped), $\sinh$ is odd (S-shaped), $\sinh(0)=0$, $\cosh(0)=1$.
Derivatives $\dfrac{d}{dx}\sinh x = \cosh x$, $\quad \dfrac{d}{dx}\cosh x = \sinh x$, $\quad \dfrac{d}{dx}\tanh x = \text{sech}^2 x$. Important: $\dfrac{d}{dx}\cosh x = +\sinh x$ (not $-\sinh x$ like for cosine!).
Integrals & Arc Length Connection $\int \sinh x\,dx = \cosh x + C$, $\quad \int \cosh x\,dx = \sinh x + C$. Arc length of $y=\cosh x$: $1+\sinh^2 x = \cosh^2 x$, so $L = \int_a^b \cosh x\,dx = \sinh b - \sinh a$.
