Numerical Integration Methods

Calc 2  ·  §17  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

Not every integral has a nice closed form. $\int_0^1 e^{-x^2}\,dx$ can't be solved symbolically. How could you estimate its value numerically using rectangles or trapezoids?
Three Rules

Midpoint Rule: $M_n=\dfrac{b-a}{n}\sum_{i=1}^n f(\bar x_i)$   ($\bar x_i$= midpoint)

Trapezoid Rule: $T_n=\dfrac{\Delta x}{2}[f(x_0)+2f(x_1)+\cdots+2f(x_{n-1})+f(x_n)]$

Simpson's Rule: $S_n=\dfrac{\Delta x}{3}[f(x_0)+4f(x_1)+2f(x_2)+\cdots+4f(x_{n-1})+f(x_n)]$

($n$ must be even for Simpson's)

▸ Show Proof

Where do these formulas come from? Each one replaces the curve on a thin strip with a shape whose area is easy to compute, then adds up the strips.

Trapezoid Rule. On one strip from $x_{i-1}$ to $x_i$ (width $\Delta x$), connect the two endpoints with a straight line. The region under that line is a trapezoid with parallel vertical sides $f(x_{i-1})$ and $f(x_i)$, so its area is

$$\frac{\Delta x}{2}\big[f(x_{i-1})+f(x_i)\big].$$

Add this over all $n$ strips. Each interior node $x_1,\dots,x_{n-1}$ is shared by two neighboring trapezoids, so it gets counted twice — that's where the $2$'s come from — while the endpoints $x_0$ and $x_n$ appear only once:

$$T_n=\frac{\Delta x}{2}\big[f(x_0)+2f(x_1)+\cdots+2f(x_{n-1})+f(x_n)\big].$$

The Midpoint Rule uses a flat-topped rectangle whose height is the curve's value at the strip's center. Simpson's Rule goes one better: it fits a parabola through each pair of strips, and working out that parabola's area produces the $1,4,2,4,\dots,4,1$ pattern — which is exactly why $n$ has to be even.

∎

▸ Common Mistake

Students try to run Simpson's Rule with an odd $n$ — say $n=5$ — and just tack on a leftover endpoint, or force the $1,4,2,4,\dots,4,1$ pattern onto an odd number of nodes. It doesn't work: Simpson's Rule pairs strips into parabola-fitted twos, so an odd $n$ leaves one strip unpaired with nowhere to go. The rule for Simpson's requires $n$ even, full stop — if a problem gives you an odd $n$ for Simpson's, either the problem intends $n+1$ subintervals or you need to switch rules.

A related slip: mixing up which interior nodes get weight $4$ versus weight $2$. The pattern always alternates starting and ending with $4$ next to the endpoints — $f(x_0)+4f(x_1)+2f(x_2)+4f(x_3)+\cdots+4f(x_{n-1})+f(x_n)$ — so miscounting the index parity (treating $x_2$ as a $4$ instead of a $2$) silently corrupts the whole sum.

Error Bounds

$|E_T|\leq\dfrac{K_2(b-a)^3}{12n^2}$   $|E_M|\leq\dfrac{K_2(b-a)^3}{24n^2}$

$|E_S|\leq\dfrac{K_4(b-a)^5}{180n^4}$

where $K_2=\max|f''|$, $K_4=\max|f^{(4)}|$ on $[a,b]$

▸ Geometric Argument

We won't pin down the exact constants ($12$, $24$, $180$) — those come from Taylor's theorem, which is just ahead in this course. But here's why the powers of $n$ are what they are.

Split $[a,b]$ into $n$ strips, each of width $h=\dfrac{b-a}{n}$. On a single strip, the straight-line (trapezoid) estimate misses the true curve by an amount controlled by how much the curve bends — its second derivative $f''$ — times the strip width cubed. Roughly, one strip's error is proportional to $K_2\,h^3$.

There are $n$ strips, so the total error is about

$$n\cdot K_2 h^3=n\cdot K_2\left(\frac{b-a}{n}\right)^3=\frac{K_2(b-a)^3}{n^2}.$$

That's exactly the $\dfrac{1}{n^2}$ shape of the Trapezoid and Midpoint bounds. Simpson's Rule cancels not just the tilt of the error but its curvature too, leaving an error per strip proportional to $h^5$; summing $n$ of those gives $\dfrac{1}{n^4}$. This is why doubling $n$ shrinks Simpson's error by a factor of about $16$, while Trapezoid's drops only by $4$.

∎

▸ Common Mistake

Students plug $K_2 = \max|f(x)|$ — a bound on the function itself — into the error formula instead of $K_2=\max|f''(x)|$, a bound on its second derivative. The letter $K$ looks like it's just "the biggest the function gets," so it's tempting to skip differentiating. But the error bound measures how much the curve bends away from a straight line (Trapezoid/Midpoint) or a parabola (Simpson's), and bending is exactly what a derivative measures — a function can be huge in size but nearly straight (small $f''$), or small in size but sharply curved (large $f''$).

Same trap applies to Simpson's: $K_4=\max|f^{(4)}(x)|$ is a bound on the fourth derivative, not the function or its first derivative. Always differentiate the correct number of times before maximizing.

Simpson's Rule is exact for polynomials of degree $\leq 3$ and converges much faster than Trapezoid.

Worked Examples
Example 1  —  Trapezoid $n=4$
Approximate $\int_0^1 x^2\,dx$ using $T_4$.
$\Delta x=0.25$; $f(0)+2f(0.25)+2f(0.5)+2f(0.75)+f(1)=0+0.125+0.5+1.125+1=2.75$
$T_4=\frac{0.25}{2}\cdot 2.75=0.34375$ (exact: $1/3$)
Example 2  —  Simpson's $n=4$
Approximate $\int_0^1 x^2\,dx$ using $S_4$.
Same nodes; $S_4=\frac{0.25}{3}[0+4(0.0625)+2(0.25)+4(0.5625)+1]$
$=\frac{0.25}{3}\cdot 4=0.3\overline{3}$ — exact! ✓
Example 3  —  Error bound
How large must $n$ be for $|E_T|\leq 0.001$ for $\int_1^2 \frac{1}{x}\,dx$?
$f''=2/x^3$; $K_2=2$ on $[1,2]$.
$\frac{2}{12n^2}\leq 0.001\Rightarrow n^2\geq 167\Rightarrow n\geq 13$
Example 4  —  Midpoint rule
Approximate $\int_0^2 e^{-x}\,dx$ using $M_4$.
Midpoints: $0.25,0.75,1.25,1.75$; $\Delta x=0.5$
$\approx0.5(e^{-0.25}+e^{-0.75}+e^{-1.25}+e^{-1.75})\approx0.5(0.7788+0.4724+0.2865+0.1738)=0.8657$
