Trigonometric Substitution

Calc 2  ·  §15  ·  Christopher Halverson  ·  Fall 2026  ·  STUDENT

Integrals involving $\sqrt{a^2-x^2}$, $\sqrt{a^2+x^2}$, or $\sqrt{x^2-a^2}$ resist $u$-substitution. Can a trig identity collapse the square root?
The Three Cases

Case 1

$\sqrt{a^2 - x^2}$

Let $x = a\sin\theta$

$\sqrt{a^2-x^2} = a\cos\theta$

Case 2

$\sqrt{a^2 + x^2}$

Let $x = a\tan\theta$

$\sqrt{a^2+x^2} = a\sec\theta$

Case 3

$\sqrt{x^2 - a^2}$

Let $x = a\sec\theta$

$\sqrt{x^2-a^2} = a\tan\theta$

▸ Show Proof

All three substitutions work for the same reason: each one turns the radicand into a perfect square using a Pythagorean identity, so the square root simplifies away.

Case 1, $x=a\sin\theta$ with $1-\sin^2\theta=\cos^2\theta$:

$$\sqrt{a^2-x^2}=\sqrt{a^2-a^2\sin^2\theta}=\sqrt{a^2(1-\sin^2\theta)}=\sqrt{a^2\cos^2\theta}=a\cos\theta.$$

Case 2, $x=a\tan\theta$ with $1+\tan^2\theta=\sec^2\theta$:

$$\sqrt{a^2+x^2}=\sqrt{a^2+a^2\tan^2\theta}=\sqrt{a^2(1+\tan^2\theta)}=\sqrt{a^2\sec^2\theta}=a\sec\theta.$$

Case 3, $x=a\sec\theta$ with $\sec^2\theta-1=\tan^2\theta$:

$$\sqrt{x^2-a^2}=\sqrt{a^2\sec^2\theta-a^2}=\sqrt{a^2(\sec^2\theta-1)}=\sqrt{a^2\tan^2\theta}=a\tan\theta.$$

In each case we take $a>0$ and $\theta$ in the standard range where the resulting trig function is non-negative, so the square root is exactly that function — no absolute-value trouble.

∎
